# SAP CPI Cost Calculator

## Database Setup

This application uses PostgreSQL. You can use a free tier from any of these providers:

- [Neon](https://neon.tech) — 512MB free, no credit card
- [Supabase](https://supabase.com) — 500MB free
- [Railway](https://railway.app) — $5 credit free
- [Render](https://render.com) — 90 days free PostgreSQL

### Configuration

Copy `.env.example` to `.env` and set your `DATABASE_URL`:

```env
DATABASE_URL=postgresql://user:password@host:5432/dbname
```
